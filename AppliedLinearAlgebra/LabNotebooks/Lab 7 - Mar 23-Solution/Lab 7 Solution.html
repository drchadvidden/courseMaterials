Recursion introduction

In programming, recursion is a method for solving a problem where the solution depends on smaller instances of the same problem. For example, factorials are computed as n!=n(n−1)(n−2)…3⋅2⋅1 n! = n(n-1)(n-2)\dots 3 \cdot 2 \cdot 1 n!=n(n−1)(n−2)…3⋅2⋅1 This problem can be rephrased in terms of itself as n!=n(n−1)! n! = n(n-1)! n!=n(n−1)! By building a function to compute n!n!n! which calls itself to compute (n−1)!(n-1)!(n−1)!, recursion can be implemented. Though, the function will need to know when to stop calling itself (called a base case) and collect all computations. For the n!n!n! example, the base case would be when the function arrives at 1!=11!=11!=1.

myDet <- function(A){
    if (nrow(A)==2){
        det <- A[1,1]*A[2,2]-A[1,2]*A[2,1]
        return(det)
    } else {
        det <- 0
        top_row <- A[1,]
        A <- A[-1,]
        for (i in 1:ncol(A)){
            Ai <- A
            Ai <- Ai[,-i]
            det <- det+(-1)^(i+1)*top_row[i]*myDet(Ai)
        }
        return(det)
    }
}

print("EXAMPLE 1")
A <- matrix(c(1,2,3,4), nrow=2, byrow=TRUE)
print(A)
print(myDet(A))
