Explanation

Recall the inverse of square matrix AAA, written A−1A^{-1}A−1, is defined as the matrix such that

AA−1=A−1A=I.A A^{-1} = A^{-1} A = I.AA−1=A−1A=I.

We can compute the iiith column of A−1A^{-1}A−1 by solving the linear system

Ax⃗=e⃗iA \vec{x} = \vec{e}_iAx=ei​

for e⃗i\vec{e}_iei​ the iiith unit basis vector.

invFunc <- function(A){
    
    if (nrow(A) != ncol(A)){
        return("ERROR: Matrix must be square to be invertible. You are crazy!")
    }

    Ainv <- matrix(0, nrow=nrow(A), ncol=ncol(A)) # inverse matrix to compute
    ident <- diag(1, nrow(A))
    
    for (i in c(1:nrow(A))){
        # compute the ith column of A^{-1}
        ei <- ident[,i]
        x <- solve(A,ei)
        Ainv[,i] <- x
    }
    
    return(Ainv)
}
